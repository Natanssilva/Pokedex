        ${Endereço(IP)}/ ${path = caminho de identificação do recurso}

IP: https://pokeapi.co  (dominio)
    http://127.0.0.1:3000  ( a mesma coisa só que em forma de porta local)

Request Method: GET | POST | PUT | DELETE | PATCH    (varios tipos)
-GET na URL do pokeapi/pokemon: buscar
-POST : Inserir
-PUT : Atualizar o pokemon
-DELETE : deletar o pokemon

!Toda requisição para um recurso, o método diz oque a gente quer fazer sobre esses recursos --> Principalmente em APIs REST

name = 1
